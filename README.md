# Linux Console File Explorer (C++17 | WSL)

##  Overview

This project is a *console-based File Explorer* implemented using *C++17* and designed to run inside the *Linux environment (WSL on Windows 11)*.
It replicates many commonly used Linux file-system commands (like ls, cd, cp, mv, rm, chmod, etc.) inside a single C++ application.

The program enables users to browse directories, create/copy/delete files, manage permissions, and search within the Linux filesystem â€” all through a custom menu-driven console.

---

##  Key Features

*  Directory navigation (cd, pwd)
*  File + directory listing (ls) with:

  * Type
  * Owner info
  * Permissions
  * Size
  * Last modification time
*  Tree view (tree)
*  File manipulation

  * Create (touch)
  * Copy (cp)
  * Move/Rename (mv)
  * Delete (rm)
*  Directory manipulation

  * Create (mkdir)
  * Recursive delete (rmdir)
*  Recursive search (search)
*  Permissions

  * View (perms)
  * Modify (chmod <octal> <path>)
*  File preview (open)
*  Built-in help command
